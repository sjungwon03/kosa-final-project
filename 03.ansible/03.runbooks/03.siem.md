# SIEM

**스택**
- Wazuh Manager: 보안 이벤트 수집·분석·알림
- Wazuh Agent: 각 VM에 설치 (common role에서 자동 배포)

**구성**
- siem-01: 172.16.30.85

## SIEM 구축 (설치)

> 최초 1회 실행

**배포**
```bash
# [컨트롤 노드] Terraform VM 생성
bash ~/workspace/terraform/02-run.sh prod apply services siem-01

# [컨트롤 노드] Ansible 서비스 구성
ANSIBLE_CONFIG=~/workspace/ansible/ansible.cfg \
  ansible-playbook \
  -i ~/workspace/ansible/inventories/prod/hosts \
  ~/workspace/ansible/playbooks/siem.yml
```

**검증**
```bash
# Ansible 접속 확인
ANSIBLE_CONFIG=~/workspace/ansible/ansible.cfg \
  ansible -i ~/workspace/ansible/inventories/prod/hosts \
  siem -m ping

# 서비스 상태 확인 (siem-01에서 실행)
systemctl status wazuh-manager

# API 응답 확인
curl -k https://172.16.30.85:55000
```
