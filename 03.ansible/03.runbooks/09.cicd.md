# CI/CD

**스택**
- Gitea: Git 저장소 (인프라 CI/CD)
- GitLab Runner: 파이프라인 실행 (Terraform/Ansible 자동화)

**구성**
- cicd-01: 172.16.30.55 (cicd.mgmt.local)

## CI/CD 구축 (설치)

> 최초 1회 실행

**배포**
```bash
# [컨트롤 노드] Terraform VM 생성
bash ~/workspace/terraform/02-run.sh prod apply services cicd-01

# [컨트롤 노드] Ansible 서비스 구성
ANSIBLE_CONFIG=~/workspace/ansible/ansible.cfg \
  ansible-playbook \
  -i ~/workspace/ansible/inventories/prod/hosts \
  ~/workspace/ansible/playbooks/cicd.yml
```

**검증**
```bash
# Ansible 접속 확인
ANSIBLE_CONFIG=~/workspace/ansible/ansible.cfg \
  ansible cicd \
  -i ~/workspace/ansible/inventories/prod/hosts \
  -m ping

# 서비스 상태 확인 (cicd-01에서 실행)
systemctl status gitea

# 브라우저 접속
# http://172.16.30.55:3000
```
