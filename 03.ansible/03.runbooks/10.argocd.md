# ArgoCD

**스택**
- ArgoCD: GitOps 기반 K8s CD (웹앱 서비스 자동 배포)

**구성**
- K8s 클러스터 내부 배포 (namespace: argocd)
- GitLab 저장소 연동 → K8s 매니페스트 자동 동기화

## ArgoCD 구축 (설치)

> 최초 1회 실행

**배포**
```bash
# [컨트롤 노드] Ansible 서비스 구성
ANSIBLE_CONFIG=~/workspace/ansible/ansible.cfg \
  ansible-playbook \
  -i ~/workspace/ansible/inventories/prod/hosts \
  ~/workspace/ansible/playbooks/argocd.yml
```

**검증**
```bash
# ArgoCD 서버 상태 확인
kubectl -n argocd get pods

# 브라우저 접속
# https://<argocd-svc-ip>
```
