## VM Creation Process

1. Terraform이 템플릿 9001에서 Full Clone
2. Cloud-init으로 설정:
   - User: kosa / Password: kosa1004
   - VLAN: DMZ(20) 또는 Internal(30)
   - IP: Static (172.16.20.x 또는 172.16.30.x)
   - Gateway: 각 VLAN Gateway
   - DNS: 8.8.8.8, 8.8.4.4

3. Ansible이 SSH로 접속 (kosa/kosa1004)
4. Percona, HAProxy, ProxySQL 설치
